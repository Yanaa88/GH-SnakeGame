# GH-SnakeGame

Проста реализация на класическата Snake игра.

Файлове
- SnakeGame.html — основна страница (зарежда `styles.css` и `game.js`)
- styles.css — стилове за оформяне на играта
- game.js — логика на играта (движение, мишки, колизии)

Как да стартираш локално

1) Най-прост (просто отвори браузър):

	Отвори `SnakeGame.html` чрез двойно кликване в Explorer.

2) Използване на вграден Python HTTP сървър (препоръчано за правилни CORS/пътеки):

```bash
python -m http.server 8000
# след това отвори http://localhost:8000/SnakeGame.html
```

3) Използване на Node (ако имаш npm):

```bash
npx serve .
# или
npx http-server .
```

Управление
- Стрелките ← ↑ → ↓ или WASD

Цел на играта
- Яжте мишки, за да расте змията.
- При удар в границите или в себе си — играта приключва.

Забележки за разработчици
- Можете да промените размера на полето в `game.js` чрез променливата `GRID`.
- Ако желаете, добавя `package.json` със скрипт за стартиране — отговорете "да".
